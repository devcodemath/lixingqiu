Python Set and Python Booleans</h2><p>So far, we have learned about various <a href="https://data-flair.training/blogs/python-variables-and-data-types/">data types in Python</a>. We dealt with <a href="https://data-flair.training/blogs/python-strings/">strings</a>, <a href="https://data-flair.training/blogs/python-number-types-conversion/">numbers</a>, <a href="https://data-flair.training/blogs/python-lists-examples/">lists</a>, <a href="https://data-flair.training/blogs/python-tuples-syntax-examples/">tuples</a>, and <a href="https://data-flair.training/blogs/python-dictionaries/">dictionaries</a>. After that, we will move onto <a href="https://data-flair.training/blogs/python-functions/">python functions</a> in further lessons.</p><div id="attachment_5772" style="width: 1210px" class="wp-caption aligncenter"><a href="https://d2h0cx97tjks2p.cloudfront.net/wp-content/uploads/sites/2/2018/01/Python-Sets-and-Booleans.jpg"><img  class="wp-image-5772"  src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" data-src="https://d2h0cx97tjks2p.cloudfront.net/wp-content/uploads/sites/2/2018/01/Python-Sets-and-Booleans.jpg"  alt="Python Sets and Booleans" width="1200" height="628" data-srcset="https://d2h0cx97tjks2p.cloudfront.net/wp-content/uploads/sites/2/2018/01/Python-Sets-and-Booleans.jpg 802w, https://d2h0cx97tjks2p.cloudfront.net/wp-content/uploads/sites/2/2018/01/Python-Sets-and-Booleans-150x79.jpg 150w, https://d2h0cx97tjks2p.cloudfront.net/wp-content/uploads/sites/2/2018/01/Python-Sets-and-Booleans-300x157.jpg 300w, https://d2h0cx97tjks2p.cloudfront.net/wp-content/uploads/sites/2/2018/01/Python-Sets-and-Booleans-768x402.jpg 768w" data-sizes="(max-width: 1200px) 100vw, 1200px" /></a><p class="wp-caption-text">Python Sets and Booleans</p></div><h2>2. Sets in Python</h2><p>First, we focus on Python sets. A set in Python holds a sequence of values. It is sequenced but does not support indexing. We will understand that as we get deeper into the article with the Python set Examples.</p><h3>a. Creating a Python Set</h3><p>To declare a set, you need to type a sequence of items separated by commas, inside curly braces. After that, assign it to a Python variable.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; a={1,3,2}</pre><p>As you can see, we wrote it in the order 1, 3, 2. In point b, we will access this set and see what we get back.</p><p>A set may contain values of different types.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; c={1,2.0,'three'}</pre><p><strong style="font-family: Verdana, Geneva, sans-serif">1. Duplicate Elements</strong></p><p>A set also cannot contain duplicate elements. Lets try adding duplicate elements to another set, and then access it in point b.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; b={3,2,1,2}</pre><p><strong style="font-family: Verdana, Geneva, sans-serif">2. Mutability</strong></p><p>A set is mutable, but may not contain mutable items like a list, set, or even a dictionary.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; d={[1,2,3],4}</pre><p>Traceback (most recent call last):</p><p>File &#8220;&lt;pyshell#9&gt;&#8221;, line 1, in &lt;module&gt;</p><p>d={[1,2,3],4}</p><p>TypeError: unhashable type: &#8216;list&#8217;</p><p>As we discussed, there is no such thing as a nested Python set.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; d={{1,3,2},4}</pre><p>Traceback (most recent call last):</p><p>File &#8220;&lt;pyshell#10&gt;&#8221;, line 1, in &lt;module&gt;</p><p>d={{1,3,2},4}</p><p>TypeError: unhashable type: &#8216;set&#8217;</p><p><strong style="font-family: Verdana, Geneva, sans-serif">3. The Python set() function</strong></p><p>You can also create a set with the set() function.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; d=set()
&gt;&gt;type(d)</pre><p>&lt;class set&gt;</p><p>This creates an empty set object. Remember that if you declare an empty set as the following code, it is an empty dictionary, not an empty set. We confirm this using the type() function.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; d={}
&gt;&gt;&gt; type(d)</pre><p>&lt;class &#8216;dict&#8217;&gt;</p><p>The set() function may also take one argument, however. It should be an iterable, like a list.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; d=set([1,3,2])</pre><h3>b. Accessing a Set in Python</h3><p>Since sets in Python do not support indexing, it is only possible to access the entire set at once. Lets try accessing the sets from point a.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; a</pre><p>{1, 2, 3}</p><p>Did you see how it reordered the elements into an ascending order? Now lets try accessing the set c.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; c</pre><p>{1, 2.0, &#8216;three&#8217;}</p><p>Finally, lets access set b.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; b</pre><p>{1, 2, 3}</p><p>As you can see, we had two 2s when we declared the set, but now we have only one, and it automatically reordered the set.</p><p>Also, since sets do not support indexing, they cannot be sliced. Lets try slicing one.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; b[1:]</pre><p>Traceback (most recent call last):</p><p>File &#8220;&lt;pyshell#26&gt;&#8221;, line 1, in &lt;module&gt;</p><p>b[1:]</p><p>TypeError: &#8216;set&#8217; object is not subscriptable</p><p>As you can see in the error, a set object is not subscriptable.</p><h3>c. Deleting a Set</h3><p>Again, because a set isnt indexed, you cant delete an element using its index. So for this, you must use one the following methods. A method must be called on a set, and it may alter the set. For the following examples, lets take a set called numbers.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers={3,2,1,4,6,5}
&gt;&gt;&gt; numbers</pre><p>{1, 2, 3, 4, 5, 6}</p><p><strong style="font-family: Verdana, Geneva, sans-serif">1. discard()</strong></p><p>This method takes the item to delete as an argument.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.discard(3)
&gt;&gt;&gt; numbers</pre><p>{1, 2, 4, 5, 6}</p><p>As you can see in the resulting set, the item 3 has been removed.</p><p><strong style="font-family: Verdana, Geneva, sans-serif">2. remove()</strong></p><p>Like the discard() method, remove() deletes an item from the set.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.remove(5)
&gt;&gt;&gt; numbers</pre><p>{1, 2, 4, 6}</p><ul><li><strong>discard() vs remove()-</strong></li></ul><p>These two methods may appear the same to you, but theres actually a difference. If you try deleting an item that doesnt exist in the set, discard() ignores it, but remove() raises a KeyError.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.discard(7)
&gt;&gt;&gt; numbers</pre><p>{1, 2, 4, 6}</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.remove(7)</pre><p>Traceback (most recent call last):</p><p>File &#8220;&lt;pyshell#37&gt;&#8221;, line 1, in &lt;module&gt;</p><p>numbers.remove(7)</p><p>KeyError: 7</p><p><strong style="font-family: Verdana, Geneva, sans-serif">3. pop()</strong></p><p>Like on a dictionary, you can call the pop() method on a set. However, here, it does not take an argument. Because a set doesnt support indexing, there is absolutely no way to pass an index to the pop method. Hence, it pops out an arbitrary item. Furthermore, it prints out the item that was popped.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.pop()</pre><p>1</p><p>Lets try popping anot/her element.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.pop()</pre><p>2</p><p>Lets try it on another set as well.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; {2,1,3}.pop()</pre><p>1</p><p><strong style="font-family: Verdana, Geneva, sans-serif">4. clear()</strong></p><p>Like the pop method(), the clear() method for a dictionary can be applied to a Python set as well. It empties the set in Python.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.clear()
&gt;&gt;&gt; numbers</pre><p>set()</p><p>As you can see, it denoted an empty set as set(), not as {}.</p><h3>d. Updating a Set</h3><p>As we discussed, a Python set is mutable. But as we have seen earlier, we cant use indices to reassign it.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers={3,1,2,4,6,5}
&gt;&gt;&gt; numbers[3]</pre><p>Traceback (most recent call last):</p><p>File &#8220;&lt;pyshell#56&gt;&#8221;, line 1, in &lt;module&gt;</p><p>numbers[3]</p><p>TypeError: &#8216;set&#8217; object does not support indexing</p><p>So, we use two methods for this purpose- add() and update(). We have seen the update() method on tuples, lists, and strings.</p><p><strong style="font-family: Verdana, Geneva, sans-serif">1. add()</strong></p><p>It takes as argument the item to be added to the set.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.add(3.5)
&gt;&gt;&gt; numbers</pre><p>{1, 2, 3, 4, 5, 6, 3.5}</p><p>If you add an existing item in the set, the set remains unaffected.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.add(4)
&gt;&gt;&gt; numbers</pre><p>{1, 2, 3, 4, 5, 6, 3.5}</p><p><strong style="font-family: Verdana, Geneva, sans-serif">2. update()</strong></p><p>This method can add multiple items to the set at once, which it takes as arguments.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers.update([7,8],{1,2,9})
&gt;&gt;&gt; numbers</pre><p>{1, 2, 3, 4, 5, 6, 3.5, 7, 8, 9}</p><p>As is visible, we could provide a list and a set as arguments to this. This is because this is different than creating a set.</p><h3>e. Functions on Sets</h3><p>A function is something that you can apply to a Python set, and it performs operations on it and returns a value. Lets talk about some of the functions that a set supports.</p><p>Well take a new set for exemplary purposes.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; days={'Monday','Tuesday','Wednesday','Thursday','Friday','Saturday','Sunday'}</pre><p><strong style="font-family: Verdana, Geneva, sans-serif">1. len()</strong></p><p>The len() function returns the length of a set. This is the number of elements in it.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; len(days)</pre><p>7</p><p><strong style="font-family: Verdana, Geneva, sans-serif">2. max()</strong></p><p>This function returns the item from the set with the highest value.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; max({3,1,2})</pre><p>3</p><p>We can make such a comparison on strings as well.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; max(days)</pre><p>&#8216;Wednesday&#8217;</p><p>The Python function returned Wednesday because W has the highest ASCII value among M, T, W, F, and S.</p><p>But we cannot compare values of different types.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; max({1,2,'three','Three'})</pre><p>Traceback (most recent call last):</p><p>File &#8220;&lt;pyshell#69&gt;&#8221;, line 1, in &lt;module&gt;</p><p>max({1,2,&#8217;three&#8217;,&#8217;Three&#8217;})</p><p>TypeError: &#8216;&gt;&#8217; not supported between instances of &#8216;str&#8217; and &#8216;int&#8217;</p><p><strong style="font-family: Verdana, Geneva, sans-serif">3. min()</strong></p><p>Like the max() function, the min() function returns the item in the Python set with the lowest value.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; min(days)</pre><p>&#8216;Friday&#8217;</p><p>This is because F has the lowest ASCII value among M, T, W, F, and S.</p><p><strong style="font-family: Verdana, Geneva, sans-serif">4. sum()</strong></p><p>The sum() functionin Python set returns the arithmetic sum of all the items in a set.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; sum({1,2,3})</pre><p>6</p><p>However, you cant apply it to a set that contains strings.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; sum(days)</pre><p>Traceback (most recent call last):</p><p>File &#8220;&lt;pyshell#72&gt;&#8221;, line 1, in &lt;module&gt;</p><p>sum(days)</p><p>TypeError: unsupported operand type(s) for +: &#8216;int&#8217; and &#8216;str&#8217;</p><p><strong style="font-family: Verdana, Geneva, sans-serif">5. any()</strong></p><p>This function returns True even if one item in the set has a Boolean value of True.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; any({0})</pre><p>False</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; any({0,'0'})</pre><p>True</p><p>It returns True because the string 0 has a Boolean value of True.</p><p><strong style="font-family: Verdana, Geneva, sans-serif">6. all()</strong></p><p>Unlike the any() function, all() returns True only if all items in the Python set have a Boolean value of True. Otherwise, it returns False.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; all({0,'0'})</pre><p>False</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; all(days)</pre><p>True</p><p><strong style="font-family: Verdana, Geneva, sans-serif">7. sorted()</strong></p><p>The sorted() function returns a sorted python set to list. It is sorted in ascending order, but it doesnt modify the original set.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; numbers={1, 2, 3, 4, 5, 6, 3.5}

</pre><p>{1, 2, 3}</p><p>As you can see, it did not alter set1.         print(i)</pre><p>1</p><p>2</p><p>3</p><p>As you can see, even though we had the order as 1,3,2, it is printed in ascending order.</p><h3>i. Python Booleans</h2><p>Finally, lets discuss Booleans. A Boolean is another data type that Python has to offer.</p><h3>a. Also note that what was once a set, we have reassigned a</p><p>Boolean to it.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; type('True')</pre><p>&lt;class &#8216;str&#8217;&gt;</p><h3>c. The bool() function</h3><p>Like we have often seen earlier, the bool() function converts another value into the Boolean type.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool('Wisdom')</pre><p>True</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool([])</pre><p>False</p><h3>d. Boolean Values of Various Constructs</h3><p>Different values have different equivalent Boolean values. In this example, we use the bool() Python set function to find the values.</p><p>For example, 0 has a Boolean value of False.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool(0)</pre><p>False</p><p>1 has a Boolean value of True, and so does 0.00000000001.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool(0.000000000001)</pre><p>True</p><p>A string has a Boolean value of True, but an empty string has False.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool(' ')</pre><p>True</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool('')</pre><p>False</p><p>In fact, any empty construct has a Boolean value of False, and a non-empty one has</p><p>True.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool(())</pre><p>False</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; bool((1,3,2))</pre><p>True</p><h3>e. Logical</strong></p><p>Finally, even the logical operators apply on Booleans.</p><pre class="EnlighterJSRAW" data-enlighter-language="null">&gt;&gt;&gt; False and True</pre><p>False</p><p>This was all about the article on Python set and booleans.</p><h2>4.
